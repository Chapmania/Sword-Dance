# Sword Dance
## 概要
これは、ライトセーバーを振り回したい！！という人類なら誰しも持ったことのあるであろう願望を叶えるべく作られたVRゲームです。開発デバイスはOculus Rift(無印)。試してないけどViveとかOculus Questとかでも動くはず。知らんけど。
